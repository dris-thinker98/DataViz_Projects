This Data Visualization Project was based on the Coursera Guided Project called "Data Visualization with Plotly Express" offered by the Coursera Project Network.

The folder contains two files mainly:-   \
i) The Jupyter Notebook (Plotly_DataViz.ipynb): The jupyter notebook python 3 code used for the interactive visualizations using gapminder dataset and plotly.express \
ii) The pdf file: To show the type of visualizations generated by the jupyter notebook.
